# Security

### <Definition>
* **English (Source 1 - Wikipedia):** "Security is a composite of confidentiality, integrity and availability."
* **English (Source 2 - NIST):** "An abstraction representing the basic properties or characteristics of an entity with respect to safeguarding information."
* **Thai:** Security ใน Quality Attributes หมายถึง ความสามารถของระบบในการปกป้องข้อมูลและทรัพยากรจากการเข้าถึงที่ไม่ได้รับอนุญาต พร้อมทั้งรับประกันว่าผู้ใช้งานที่มีสิทธิ์ยังคงสามารถเข้าถึงบริการได้อย่างถูกต้อง โดยมีเป้าหมายหลักในการต้านทาน ป้องกัน และฟื้นฟูระบบจากการโจมตีที่เป็นอันตรายครับ

### <Explanation>
* "คุณลักษณะที่ทำให้ระบบสามารถปกป้องข้อมูล ทรัพยากร และบริการจากการเข้าถึง การแก้ไข หรือการโจมตีโดยไม่ได้รับอนุญาต นอกจากนี้ยังรวมถึงการตรวจสอบสิทธิ์ความถูกต้อง (Authentication) และการกำหนดสิทธิ์การใช้งาน (Authorization) เพื่อให้มั่นใจว่าโครงสร้างพื้นฐานของระบบและข้อมูลที่สำคัญจะไม่ถูกเปิดเผยหรือถูกดัดแปลงโดยผู้ไม่ประสงค์ดี", **(ChatGPT 4o)**
* "ความสามารถของระบบในการปกป้องข้อมูลและทรัพยากรจากการเข้าถึงที่ไม่ได้อนุญาต การโจมตี หรือการดัดแปลงแก้ไข ในขณะที่ยังคงต้องยอมให้ผู้ใช้ที่มีสิทธิ์จริง ๆ (Authorized users) เข้าใช้งานได้อย่างราบรื่น การมีระบบรักษาความปลอดภัยที่แข็งแกร่งจะช่วยสร้างความน่าเชื่อถือ ลดความเสี่ยงจากการถูกเจาะระบบ และทำให้ระบบทำงานได้อย่างต่อเนื่อง", **(Google Gemini)**
